상세 설명: 참여인원을 선택하고 중복 없이 두명씩 뽑을 수 있는 기능

핵심 기능 2: 술 룰렛
상세 설명: 슬롯머신을 기본으로 하여, 버튼을 누르면 이름, 지목인원수, 잔 수 가 나오는 기능

핵심 기능 3: 토너먼트 앱
상세 설명: 토너먼트 기능
